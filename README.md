# StreamVerse 🎬

StreamVerse is a web application that allows users to explore and discover movies and TV shows.
It integrates external movie APIs and a backend service to fetch, display, and manage media information in a clean and interactive interface.

The project demonstrates frontend development, API integration, and backend services.


## 🚀 Features

- Browse and discover movies and TV shows
- Fetch real-time media data using external APIs
- Dynamic movie and TV show listings
- Responsive UI for seamless browsing
- Backend integration for data handling and authentication


## 🛠 Technologies Used

Frontend
- HTML
- CSS
- JavaScript

Backend
- Supabase
- APIs
- TMDB API (The Movie Database)
- TV Show APIs


## 📚 Project Overview

StreamVerse allows users to search and browse trending or popular media content.
The application fetches movie and TV show data from external APIs and dynamically displays them in the interface.

This project helped strengthen understanding of:

- API integration in web applications

- Fetching and rendering dynamic data

- Backend services using Supabase

- Building responsive web interfaces
